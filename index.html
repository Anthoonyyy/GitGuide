GRAND PROJET
MON PC - empty

  1. First off, let's go to the github repository
  2. From here we need to go to "Code" and then copy the lien (either SSH or HTTPS). Then open BASH in the desired folder and copy the entire repository. The command is // git clone https://github.com/Leerlandais/GitGuide.git // Click here to see what happens when you do this
  3. Now that we have the files on our PC, we can work on them. Before we do that, we might as well add somewhere to put our files. The command is // git remote add origin main // Click here to see what happens when you do this
  4. With all that done we can now work on the changes. In the Grand Project, you've been put in charge of images. So add them. Click here to see what happens when you do this
  5. Now that we've done the necessary work, we need to upload it to git so it's accessible to others. To do that, you return to BASH and prepare your files // git add . // What happens here is GIT checks for differences between the files on your PC with the Upstream version and flags any changes made
  6. The next step is to prepare the files for sending, where we include a description of changes. The command is // git commit -m"Your Message // Your changes are now ready to be sent...
  7. ...and this is why we made origin. Because we don't own the Grand Project, we need first to send it to git using this command // git push origin main // As usual, click here to see what happens when you do this
  8. Now that the changes are uploaded to git, we need to tell the Project owner that the work is done. Return to Git and use the Pull Request. Click here to see what happens when you do this
  9. The owner can now see your changes and accept them (merging them into the Grand Project). Click here to see what happens
  10. Or he can refuse them. Click here to see what happens when you do this
